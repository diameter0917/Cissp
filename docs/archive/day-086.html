考點：分類分級與角色

Exam: Classification & Roles
01 — 考前重點濃縮

D2 後期。本封考分類分級與角色判斷(owner vs custodian、controller vs processor)——D2 最高頻考點。題目英文＋中文解析。

決定分類＝Data Owner(業務,非 IT);執行保護＝Custodian(IT)。
決定處理目的＝Controller;代為處理＝Processor。
分級『數量』不重要,重點是『每級對應的保護要求(handling)』。
02 — 角色速記
角色負責
Data Owner決定分類、核准存取(業務主管)
Data Custodian執行備份/加密/設定(IT)
Controller決定處理目的與方式
Processor依指示處理(雲端/外包)
03 — 關鍵名詞（中英）
Data Owner（資料擁有者）— 決定分類與存取(業務,不可下授當責)。
Data Custodian（保管者）— 依擁有者要求執行技術保護。
Controller/Processor（控管/處理者）— 決定目的 vs 照指示處理。
Labeling（標示）— 系統可讀的分類標記。
Handling Requirements（處理要求）— 各級的儲存/傳輸/銷毀規定。
04 — 常見陷阱

Owner vs Custodian:決定/核准＝Owner(業務);執行＝Custodian(IT)。看到『決定分類』選 Owner。

Controller vs Owner:Owner 是內部角色;Controller 是法律主體(GDPR 語境)。

誰決定分類:永遠是業務 Owner,不是 IT、不是資安長。

05 — 🎯 實戰考題
QUESTION 1

Who is PRIMARILY responsible for determining the classification level of a dataset?

（誰主要負責決定一份資料集的分類等級?）

A. The data custodian 保管者
B. The data owner 資料擁有者
C. The security administrator 安全管理員
D. The end user 終端使用者
✅ 解答與解析

正解：B。Data Owner(業務擁有者)最了解資料價值與用途,負責決定分類等級;這是當責、不可下授。

為何其他選項錯：
  • A：Custodian(IT)只負責『執行』擁有者所定的保護,不決定分類。
  • C：安全管理員提供技術支援,非分類決策者。
  • D：使用者依政策使用資料,無分類決定權。
QUESTION 2

An IT administrator performs daily backups and configures encryption for a financial database according to policy. This person is acting as the:

（一位 IT 管理員依政策為財務資料庫執行每日備份並設定加密。此人扮演的角色是:）

A. Data owner 資料擁有者
B. Data custodian 保管者
C. Data controller 控管者
D. Data subject 當事人
✅ 解答與解析

正解：B。依擁有者所定要求執行備份、加密等技術保護＝Data Custodian(保管者)。

為何其他選項錯：
  • A：擁有者『決定』分類與保護需求,不親自執行技術操作。
  • C：Controller 是 GDPR 中決定處理目的的法律主體,非此情境。
  • D：當事人是資料所描述的本人。
QUESTION 3

A hospital uses an external billing company that processes patient data strictly according to the hospital's instructions. Under privacy law, the billing company is the:

（一家醫院委託外部帳務公司,嚴格依醫院指示處理病患資料。隱私法下,該帳務公司是:）

A. Data controller 控管者
B. Data processor 處理者
C. Data owner 資料擁有者
D. Joint controller 共同控管者
✅ 解答與解析

正解：B。帳務公司依醫院指示處理資料、不自行決定目的＝Processor;醫院決定目的＝Controller。

為何其他選項錯：
  • A：決定處理目的的是醫院(Controller),非帳務公司。
  • C：Owner 是內部角色,此題問隱私法的法律角色。
  • D：共同控管需『共同決定目的』,此處帳務公司只是照指示,非共同決定。
06 — (ISC)² 選最佳答案思路

角色題口訣:『決定/核准』→ Owner(業務)、Controller(法律);『執行/照做』→ Custodian(IT)、Processor(外包)。

看到『誰決定分類』一律 Data Owner;『代為處理個資的廠商』一律 Processor。

✏️ 一句話總結 + 自我檢測（點開看解答）

一句話：決定分類＝Owner(業務)、執行＝Custodian(IT);決定目的＝Controller、照做＝Processor。

Q1 . 決定資料分類等級的角色？
→ Data Owner(業務擁有者)。

Q2 . 依指示為醫院處理病患資料的外部公司是？
→ Processor(處理者)。

Q3 . 執行備份與加密設定的 IT 人員是什麼角色？
→ Data Custodian(保管者)。
